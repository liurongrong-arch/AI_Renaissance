### 牛市逐条逻辑

| 专家 | 权重 | 核心理由 |
|:---|:---:|:---|
| 技术面 | 1.3 | 牛市=趋势市场，趋势跟踪策略在此环境收益最优（LuxAlgo/AQR） |
| 资金流 | 1.2 | 牛市由增量资金驱动，资金流入验证趋势强度 |
| 产业分析 | 1.05 | 行业轮动是 alpha 来源（林奇方法论/A股实证） |
| 宏观面 | 1.0 | 宏观变化已计入价格，角色转为监测政策拐点 |
| 财务面 | 0.7 | 价值因子在牛市收益接近零（QuantPedia） |
| 舆情组 | 0.7 | 双向噪声：过度乐观+过早警告 |
| 风险面 | 0.5 | 风险溢价压缩，误报率升高（CEPR） |

### 熊市逐条逻辑

| 专家 | 权重 | 核心理由 |
|:---|:---:|:---|
| 风险面 | 1.6 | 风控从辅助升级为核心（CEPR/巴菲特原则） |
| 财务面 | 1.2 | 价值因子在熊市收益高度正显著（QuantPedia） |
| 产业分析 | 1.10 | 行业防御属性决定抗跌能力（护城河理论） |
| 宏观面 | 1.1 | 政策拐点是熊市结束最早信号（2008/2020验证） |
| 资金流 | 0.9 | 信号价值不对称，需区分流出类型 |
| 舆情组 | 0.65 | 负面信息放大2-3倍（行为金融学/AAII） |
| 技术面 | 0.5 | 动量崩溃风险集中于熊市反弹期（AQR） |

## 仓位公式

| 场景 | 系数 | 上限 | 公式 | 设计逻辑 |
|:---|:---:|:---:|:---|:---|
| 牛市 | 0.60 | 0.40 | `min(conf×0.60, 0.40)` | 趋势持续性强，可适度提高但保留安全边际 |
| 熊市 | 0.25 | 0.15 | `min(conf×0.25, 0.15)` | 亏损不可逆性 > 少赚机会成本 |
| 震荡市 | 0.35 | 0.20 | `min(conf×0.35, 0.20)` | 方向不明时重仓导致持续小亏损累积 |

## match_signals 评分规则

| 场景 | 条件 | 聚合 | 阈值 |
|:---|:---|:---|:---:|
| 牛市 | 宏观 bullish、风控 bullish、信号偏多 | `(c1+c2+c3)/3` | ≥0.5 |
| 熊市 | 宏观 bearish、风控 bearish、信号偏空 | `(c1+c2+c3)/3` | ≥0.5 |
| 震荡市 | 宏观 neutral、多空均衡、宏风控冲突 | top-2 平均 | ≥2 条件 >0.5 |

每条件评分：

| 情况 | 分数 |
|:---|:---|
| 方向匹配 + conf ≥ 0.5 | conf |
| 方向匹配 + conf < 0.5 | conf × 0.6 |
| 方向不匹配或无信号 | 0 |
| 风控缺失（部分分） | 0.3 |

## 校准参数

| 参数 | 值 | 说明 |
|:---|:---|:---|
| α (信任度) | 0.3 | 70% 理论锚定 + 30% 实证数据 |
| scale | 1.6 | 0-1 准确率 → 权重范围映射系数 |
| band | ±30% | 每个权重在锚定值 ±30% 内调整 |
| 方向约束 | 不可逆 | 理论 ↑ 的不能因实证差而 ↓ |
| min_samples | 50 | 低于此值 α 自动衰减 |
| stability | <15% | 连续两期校准值波动上限 |
