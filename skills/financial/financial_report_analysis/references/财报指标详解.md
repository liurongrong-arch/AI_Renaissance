# 财报指标详解

本文档提供财报分析技能中所有核心指标的定义、计算公式、判断标准和实战解读要点。

---

## 〇、数据溯源标注规范

> **核心原则：没有来源的数据，就是幻觉的温床。**

### 标注格式

每个指标数据后必须附来源标注：

```
原始数据：[数值] ← [来源标识]
计算比率：[数值] ← 计算：[公式简写] ← [底层数据来源]
估算值：[数值] ← ⚠️估算：[估算依据] ← [参考来源]
```

### 来源标识速查

| 来源 | 标识 | 可信度 |
|------|------|--------|
| 东方财富API（巨潮公告原文） | `东财API·<三表>·<报告期>` | ★★★★★ |
| 巨潮资讯网PDF | `巨潮PDF·<公告名>·<页码>` | ★★★★★ |
| 上市公司官方公告 | `公司公告·<日期>·<标题>` | ★★★★★ |
| westock-data | `westock·<命令>·<代码>` | ★★★★ |
| neodata | `neodata·<关键词>` | ★★★★ |
| 二手财经媒体 | `⚠️<媒体名>·<日期>` | ⚠️★★ |

### 标注示例

```
# A类：原始数据
合同负债 1.795亿 ← 东财API·资产负债表·2026Q1
购建固定资产支付现金 5028万 ← 东财API·现金流量表·2026Q1

# B类：计算比率
收现比 1.77 ← 计算：销售收现2.901亿/营收1.638亿 ← 东财API·现金流量表+利润表·2026Q1
净债务率 67.0% ← 计算：(短借9.923+长借3.361+一年内到期0.865-货币资金3.253)/股东权益16.256 ← 东财API·资产负债表·2026Q1

# C类：低可信来源（必须加⚠️）
⚠️ 产能利用率约80% ← ⚠️券商研报·无法验证
⚠️ 行业合同负债均值30% ← ⚠️行业研报·2025
```

### 特殊规则

1. **同比/环比变化率**：需同时标注本期和上期数据的来源
   - `合同负债同比+166.59% ← 计算：本期1.795亿/上年同期0.673亿-1 ← 东财API·资产负债表·2026Q1+2025Q1`
2. **多个来源交叉验证同一数据**：取可信度最高的来源标注
3. **无法验证的数据**：不得作为判断依据，必须用⚠️标记并注明"无法验证"
4. **估算值**：必须写出估算公式和每个变量的来源

## 一、利润现金验证指标

### 1.1 经营现金流/归母净利润（现金利润比）

**公式：**
```
现金利润比 = 经营活动产生的现金流量净额 / 归属于母公司所有者的净利润
```

**判断标准：**

| 区间 | 评级 | 含义 |
|------|------|------|
| > 1.2 | 优秀 | 每1元利润有超1.2元现金支撑，利润质量极高 |
| 1.0 - 1.2 | 良好 | 利润有充足现金支撑 |
| 0.8 - 1.0 | 一般 | 现金略低于利润，需关注趋势 |
| 0.5 - 0.8 | 警惕 | 利润可能含水分，需深挖原因 |
| < 0.5 | 危险 | 利润严重缺乏现金支撑，纸面富贵 |

**实战要点：**
- 单期低于1不一定是问题，但连续3期以上低于1必须警惕
- 季度数据波动大，优先看年度或滚动12个月
- 某些行业天然现金利润比偏低（如重资产折旧期），需行业对比
- 如果利润暴增但现金利润比骤降，可能是通过应收/存货调节利润

### 1.2 销售收现/营收（收现比）

**公式：**
```
收现比 = 销售商品、提供劳务收到的现金 / 营业收入
```

**判断标准：**

| 区间 | 评级 | 含义 |
|------|------|------|
| > 1.15 | 优秀 | 销售不仅全部收现，还收回前期欠款 |
| 1.0 - 1.15 | 良好 | 销售基本收现，含增值税因素 |
| 0.9 - 1.0 | 一般 | 大部分收现，小部分挂账 |
| 0.8 - 0.9 | 警惕 | 大量收入未收回现金 |
| < 0.8 | 危险 | 严重依赖赊销，坏账风险大 |

**实战要点：**
- 收现比>1不完全等于回款好，因为含增值税（税率13%时1.13为正常值）
- 收现比突然下降，需检查是否放宽信用政策冲业绩
- 与同行对比：同行业内收现比持续偏低的公司，竞争力可能不足

### 1.3 应收账款变化率 vs 营收变化率

**公式：**
```
应收变化率 = (本期应收账款 - 上期应收账款) / 上期应收账款
营收变化率 = (本期营收 - 上期营收) / 上期营收
```

**判断矩阵：**

| 应收变化率 | 营收变化率 | 信号 | 解读 |
|-----------|-----------|------|------|
| ↓ | ↑ | 强积极 | 收入增长但应收下降，议价能力极强 |
| ↑（慢于营收） | ↑ | 积极 | 收入和应收同步增长，但应收增速慢 |
| ↑（快于营收） | ↑ | 警惕 | 放宽信用冲业绩，增长质量存疑 |
| ↑ | ↓ | 危险 | 收入下滑但应收增加，回款恶化 |

## 二、需求真实性验证指标

### 2.1 营运资本信号

**关键组合：**

**积极信号（需求真实）：**
- 应收账款下降 + 收现比上升 → 客户愿意提前付款，公司议价能力强
- 存货上升 + 在建工程上升 → 为未来交付备货备产，不是滞销

**消极信号（需求虚胖）：**
- 应收账款上升 + 收现比下降 → 客户拖欠，回款困难
- 存货上升 + 在建工程下降/停滞 → 可能是产品滞销积压

**中性信号（需进一步验证）：**
- 应收上升 + 存货上升 + 收现比稳定 → 可能是正常扩产备货

### 2.2 存货周转天数变化

**公式：**
```
存货周转天数 = 365 / (营业成本 / 平均存货)
平均存货 = (期初存货 + 期末存货) / 2
```

**判断：**
- 周转天数缩短 → 销售加速，需求旺盛
- 周转天数拉长 → 可能滞销或盲目生产

## 三、业绩先行指标

### 3.1 合同负债变化率

**公式：**
```
合同负债变化率 = (本期合同负债 - 上期合同负债) / 上期合同负债
```

**判断标准：**

| 变化率 | 信号 | 含义 |
|--------|------|------|
| 大幅上升（>20%） | 强积极 | 订单已转化为预收现金，未来业绩有保障 |
| 小幅上升（0-20%） | 积极 | 订单稳步增长 |
| 小幅下降（0至-20%） | 警惕 | 新订单可能减速 |
| 大幅下降（<-20%） | 危险 | 订单严重不足，未来收入可能断崖 |

**实战要点：**
- 合同负债/预收款项是业绩的先行指标，比分析师预测都靠谱
- 需区分合同负债性质：部分可能是补贴或保证金，需剔除
- 对比合同负债与下期营收的关系：合同负债大幅上升后，通常1-2个季度内营收兑现

### 3.2 预收款项 vs 预付款项（注意区分）

> ⚠️ **重要区分**：预收款和预付款是相反方向的科目，不能混淆！
> - **预收款/合同负债**（负债科目）：客户预先付给公司的钱 → 订单先行指标
> - **预付款项**（资产科目）：公司预先付给供应商的钱 → 采购备货信号

**预收款项（合同负债）变化 — 业绩先行指标（核心）**

已在3.1节详述。预收款上升 = 客户提前打款 = 订单真实 = 业绩有保障。

**预付款项变化 — 采购备货辅助信号**

**公式：**
```
预付款项变化率 = (本期预付款项 - 上期预付款项) / 上期预付款项
```

- 预付款项上升 → 公司在为采购备货提前锁价，预期需求确定
- 预付款项下降 → 可能对上游议价能力增强，也可能采购收缩

**预收款+预付款联动信号：**
- 预收款↑ + 预付款↑ → 客户付钱在先，公司备货在后，全链条活跃
- 预收款↑ + 预付款↓ → 客户积极付款，公司议价能力强不需预付
- 预收款↓ + 预付款↑ → 需警惕：公司在备货但客户未预付，可能是被动库存

## 四、产能扩张信号指标

### 4.1 在建工程/固定资产比率

**公式：**
```
在建工程占比 = 在建工程 / (在建工程 + 固定资产)
```

**判断：**
- 占比 > 20% → 公司处于较密集的产能建设期
- 占比 10%-20% → 正常扩产节奏
- 占比 < 10% → 扩产力度较小或建设接近尾声

**实战要点：**
- 在建工程需关注转固速度：长期不转固可能是在调节折旧
- 在建工程上升 + 固定资产上升 → 新产能投放在即
- 在建工程上升 + 固定资产不变 → 建设中，尚未投产

### 4.2 三指标联动判断

| 在建工程 | 固定资产 | 资本开支 | 信号 |
|---------|---------|---------|------|
| ↑ | ↑ | ↑ | 强扩张：真金白银在投未来 |
| ↑ | → | ↑ | 建设中：产能即将释放 |
| → | ↑ | ↓ | 扩产完成：进入产能释放期 |
| ↓ | → | ↓ | 收缩期：停止投资 |

## 五、扩张风险评估指标

### 5.1 净债务水平

**公式：**
```
净债务 = 短期借款 + 长期借款 + 应付债券 - 货币资金
净债务率 = 净债务 / 净资产
```

**判断标准：**

| 净债务率 | 评级 | 含义 |
|---------|------|------|
| < 0（净现金） | 安全 | 现金充裕，无还债压力 |
| 0 - 30% | 良好 | 杠杆可控 |
| 30% - 60% | 一般 | 需关注利息覆盖能力 |
| 60% - 100% | 警惕 | 杠杆较高，需密切监控 |
| > 100% | 危险 | 债务沉重，流动性风险大 |

### 5.2 借款结构分析

**公式：**
```
短债占比 = 短期借款 / (短期借款 + 长期借款)
短债变化率 = (本期短期借款 - 上期短期借款) / 上期短期借款
长债变化率 = (本期长期借款 - 上期长期借款) / 上期长期借款
```

**危险信号：**
- 短期借款暴增 → 可能资金链紧张，借短债周转
- 长期借款大增 + 资本开支未放量 → 借钱不是为了扩产，要警惕用途
- 短债占比持续上升 → 滚动融资压力增大

### 5.3 利息覆盖率

**公式：**
```
利息覆盖率 = (营业利润 + 财务费用) / 财务费用
或简化为 = EBIT / 利息支出
```

**判断标准：**

| 利息覆盖率 | 评级 |
|-----------|------|
| > 10 | 安全 |
| 5 - 10 | 良好 |
| 3 - 5 | 一般 |
| 1.5 - 3 | 警惕 |
| < 1.5 | 危险，可能无法覆盖利息 |

## 六、财务费用侵蚀指标

### 6.1 财务费用/营业利润

**公式：**
```
财务费用侵蚀度 = 财务费用 / 营业利润
```

**判断标准：**

| 侵蚀度 | 评级 | 含义 |
|--------|------|------|
| < 5% | 安全 | 利息支出对利润几乎无影响 |
| 5% - 10% | 良好 | 利息支出可控 |
| 10% - 20% | 警惕 | 利息开始侵蚀利润，需监控趋势 |
| 20% - 40% | 危险 | 利润被利息严重侵蚀 |
| > 40% | 极危 | 经营利润大部分用来还利息 |

**实战要点：**
- 在利率上行周期尤其要警惕此指标恶化
- 固定利率贷款多的公司，短期受加息影响小
- 浮动利率贷款占比高的公司，需按当前利率做压力测试

### 6.2 财务费用增速 vs 营收增速

**公式：**
```
财务费用增速 = (本期财务费用 - 上期财务费用) / 上期财务费用
营收增速 = (本期营收 - 上期营收) / 上期营收
```

**判断：**
- 财务费用增速 > 营收增速 → 杠杆开始反噬，增长被利息吃掉
- 财务费用增速 < 营收增速 → 增长质量尚可，利息可控

## 七、综合逻辑链验证

### 7.1 逻辑链完整度评估

**完整链条：** 订单真实 → 现金先回 → 应收没恶化 → 存货和在建工程抬升 → 资本开支放量 → 继续扩产兑现

**评估方法：**

| 链条状态 | 条件 | 评级 |
|---------|------|------|
| 完整 | 6个环节全部验证通过 | A级：高确定性增长 |
| 大部分完整 | 4-5个环节通过 | B级：需关注薄弱环节 |
| 部分断裂 | 2-3个环节通过 | C级：增长逻辑存疑 |
| 严重断裂 | 0-1个环节通过 | D级：增长逻辑不可信 |

### 7.2 各环节对应指标

| 环节 | 核心验证指标 | 通过条件 |
|------|------------|---------|
| 订单真实 | 合同负债同比变化 | 合同负债上升 |
| 现金先回 | 经营现金流/归母净利润 | > 1.0 |
| 应收没恶化 | 应收变化率 vs 营收变化率 | 应收增速 ≤ 营收增速 |
| 存货和在建抬升 | 存货变化 + 在建工程变化 | 双升或在建升 |
| 资本开支放量 | 购建固定资产支付现金变化 | 同比上升 |
| 扩产兑现 | 固定资产变化 + 营收变化 | 固定资产转固后营收增长 |

## 附录：行业特殊处理

以下行业需特殊处理，不完全适用本框架：

| 行业 | 特殊情况 | 建议 |
|------|---------|------|
| 银行/保险 | 资产负债结构完全不同 | 不适用，需用专门的金融行业分析框架 |
| 房地产 | 合同负债含义不同（预售房款） | 需结合交付周期分析 |
| 建筑/工程 | 应收天然大、合同负债天然高 | 需与同行对比，而非看绝对值 |
| 互联网/软件 | 轻资产，固定资产和在建工程意义小 | 重点关注用户/ARR指标替代产能指标 |
| 生物医药 | 研发投入大，利润可能长期为负 | 用研发投入替代资本开支分析 |
