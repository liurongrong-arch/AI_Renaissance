## 一、模型概述

### 1.1 CCF 定义

**Cost Conduction Factor（成本传导系数）**：衡量产业链上下游之间价格变动的传导效率。CCF>1 表示放大传导（下游比上游波动更剧烈），CCF<1 表示衰减传导（下游波动被稀释），CCF≈0 表示阻断传导（下游价格不受影响）。

### 1.2 模型适用范围

| 适用场景 | 说明 |
|---------|------|
| 原材料涨价预警 | 当InP衬底/铟价上涨时，预测对光芯片→光模块→终端的成本冲击 |
| 毛利率压力测试 | 计算各环节在涨价潮中的利润挤压/转移空间 |
| 投资 timing 判断 | 判断产业链哪一环节率先承压/率先受益 |
| 供应商谈判 | 量化成本分摊比例，建立谈判基准 |

### 1.3 产业链结构图

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  上游：InP衬底  │ ──▶ │  中游：光芯片   │ ──▶ │  下游：光模块   │ ──▶ │  终端：云厂商   │
│  (云南锗业/AXT) │     │ (Lumentum/     │     │ (中际旭创/      │     │ (Google/Amazon/ │
│               │     │  Coherent/源杰) │     │  新易盛/光迅)   │     │  Meta/字节/阿里/腾讯)│
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
     P_inp               P_chip              P_module             Q_cloud
```

## 二、各环节定价公式

### 2.1 上游：InP衬底

**定价基准**：
- 衬底价格主要由 **锗/铟原材料成本** + **晶体生长工艺成本** + **切割抛光成本** + **合理利润** 构成
- 全球InP衬底市场被住友电工、AXT、JX金属三家企业合计垄断 **90%+** 产能 🟡

**价格公式**：

```
P_inp = P_raw × (1 + r_raw→inp) + C_growth + C_polish + π
```

| 变量 | 含义 | 典型值 |
|-----|------|--------|
| P_raw | 精铟/锗原料价格 | 4,350元/kg（2026年4月） ✅ |
| r_raw→inp | 原料→衬底加工损耗率 | 15-25% 🟡 |
| C_growth | 晶体生长成本（坩埚、能耗、人工） | ~50-80美元/片（4寸） 🟡 |
| C_polish | 切割抛光成本 | ~20-30美元/片（4寸） 🟡 |
| π | 厂商利润加成 | 毛利率 30-50% 🟡 |

### 2.2 中游：光芯片

**定价公式**：

```
P_chip = P_inp × (1 + g_chip) × (1 + r_loss) + C_epi + C_fab + π_chip
```

| 变量 | 含义 | 典型值 |
|-----|------|--------|
| P_inp | InP衬底采购价 | 参见2.1 |
| g_chip | 芯片加工毛利率 | **45-75%**（高端EML） 🟡 |
| r_loss | 外延+流片良率损耗 | 20-40%（200G EML） 🔴 |
| C_epi | 外延片成本（MOCVD等） | ~30-50美元/片 🟡 |
| C_fab | 流片/封装测试成本 | ~20-40美元/颗 🟡 |
| π_chip | 品牌/技术溢价 | 高端芯片额外+20-50% 🔴 |

**关键说明**：
- 高端200G EML芯片毛利率 **>60%**（Lumentum/Coherent财报口径）🟡
- 源杰科技2025年综合毛利率 **58.11%**，数据中心产品毛利率 **72.21%** ✅
- 光芯片在光模块BOM中占比 **35-50%**（EML方案） ✅

### 2.3 下游：光模块

**定价公式**：

```
P_module = [Σ(P_chip_i × Q_i) + Σ(P_elec_j × Q_j) + C_optical + C_packaging] 
           × (1 + g_module) + π_module
```

| 变量 | 含义 | 典型值 |
|-----|------|--------|
| P_chip_i | 各类光芯片采购价 | EML $12-18/颗, VCSEL $8-15/颗 ✅ |
| Q_i | 芯片用量 | 800G DR8需8颗EML ✅ |
| P_elec_j | 电芯片（DSP等）采购价 | DSP $100-170/颗 ✅ |
| Q_j | 电芯片用量 | 800G通常1颗DSP ✅ |
| C_optical | 光学组件（透镜/耦合器等） | ~$40-60/模块 🟡 |
| C_packaging | 封装+PCB+结构件+人工 | ~$50-80/模块 🟡 |
| g_module | 组装毛利率 | **25-46%**（头部厂商） ✅ |
| π_module | 规模效应摊薄 | 量产每翻倍BOM降5-10% 🟡 |

### 2.4 终端：云厂商

**需求函数**：

```
Q_cloud = f(CAPEX_budget, GPU_cluster_scale, interconnect_topology)
```

| 变量 | 说明 |
|-----|------|
| CAPEX_budget | 年度资本开支预算 |
| GPU_cluster_scale | GPU集群规模（H100→GB200→Rubin） |
| interconnect_topology | 网络拓扑结构（影响光模块配比） |

**典型配比**（每个GPU对应光模块数量）🟡：
- H100集群：~2.5个800G模块/GPU
- GB200集群：~5-6个1.6T模块/GPU（NVLink Switch引入）
- Rubin集群：~8-10个3.2T模块/GPU（预测）

## 三、CCF（传导系数）计算体系

### 3.1 上游→中游 CCF_up

**定义**：

```
CCF_up = ΔP_chip / ΔP_inp
```

**推导过程**：

假设InP衬底价格上涨 ΔP_inp，光芯片厂商的成本变化为：

```
ΔCost_chip = ΔP_inp × (1 + r_loss) × Q_inp_per_chip
```

若芯片厂商选择 **全额传导** 成本（保持毛利率不变）：

```
P_chip' = (Cost_chip + ΔCost_chip) / (1 - g_chip)

ΔP_chip = ΔCost_chip / (1 - g_chip)
        = ΔP_inp × (1 + r_loss) × Q_inp_per_chip / (1 - g_chip)
```

因此：

```
CCF_up = (1 + r_loss) × Q_inp_per_chip / (1 - g_chip)
```

**实际参数代入**（200G EML芯片为例）：

| 参数 | 值 | 来源 |
|-----|-----|------|
| 单片4寸InP衬底可切割芯片数 | ~200-400颗 | 🟡 产业调研 |
| 良率损耗 r_loss | 30%（200G EML） | 🔴 单一来源 |
| 芯片加工毛利率 g_chip | 65%（高端EML） | ✅ 源杰科技72%、产业平均65% |
| Q_inp_per_chip（衬底面积占比） | ~1/250 | 🟡 |

```
CCF_up ≈ (1 + 0.30) × (1/250) / (1 - 0.65)
       ≈ 1.30 × 0.004 / 0.35
       ≈ 0.015
```

**结论**：上游衬底价格变动 **1%** → 中游芯片价格变动约 **0.015%**，即 **1.5%**

> 这意味着：InP衬底价格上涨对芯片成本端的直接冲击被芯片环节的高毛利率大幅稀释。但如果芯片厂商选择不传导、自行消化，则毛利率将被挤压。

### 3.2 中游→下游 CCF_down

**定义**：

```
CCF_down = ΔP_module / ΔP_chip
```

**推导过程**：

假设光芯片价格上涨 ΔP_chip，光模块厂商的成本变化为：

```
ΔCost_module = ΔP_chip × Q_chip_per_module × (1 + r_assembly_loss)
```

若模块厂商选择 **全额传导**（保持毛利率不变）：

```
P_module' = (Cost_module + ΔCost_module) / (1 - g_module)

ΔP_module = ΔCost_module / (1 - g_module)
          = ΔP_chip × Q_chip_per_module × (1 + r_assembly_loss) / (1 - g_module)
```

因此：

```
CCF_down = Q_chip_per_module × (1 + r_assembly_loss) / (1 - g_module)
```

**实际参数代入**（800G DR8 EML方案）：

| 参数 | 值 | 来源 |
|-----|-----|------|
| EML芯片用量 Q_chip | 8颗/模块 | ✅ |
| 组装良率损耗 r_assembly | 10-15% | 🟡 |
| 模块组装毛利率 g_module | 40%（头部） | ✅ 中际旭创2026Q1约46% |

```
CCF_down ≈ 8 × (1 + 0.12) / (1 - 0.40)
         ≈ 8 × 1.12 / 0.60
         ≈ 14.9
```

**结论**：中游芯片价格变动 **1%** → 下游模块价格变动约 **14.9%**

> 这意味着：光芯片涨价对模块成本端的冲击被 **大幅放大**。芯片是模块BOM的最大单一成本项，芯片涨价10% → 模块成本上涨约 **149%**（按全额传导计），模块厂商要么压缩自身毛利率，要么被迫涨价。

### 3.3 转嫁率（Pass-through Rate）

**定义**：下游厂商将成本上涨转嫁给终端客户的能力

```
转嫁率 = 实际售价涨幅 / 理论成本涨幅
       = ΔP_module_sell / ΔCost_module
```

**影响因素**：

| 因素 | 对转嫁率的影响 |
|------|---------------|
| 供需紧张程度 | 供不应求 → 转嫁率↑（当前光模块市场属此类）✅ |
| 客户集中度 | 客户集中（如NVDA占50%+）→ 转嫁率↓ 🔴 |
| 合约锁定 | 长单锁价 → 转嫁率↓（新订单生效后传导）🟡 |
| 竞争格局 | 头部集中 → 转嫁率↑（协同定价）✅ |
| 替代方案 | 硅光/LPO可替代 → 转嫁率↓ 🟡 |

**当前光模块市场转嫁率评估**（2026年Q2）：

- **供需缺口**：200G EML芯片缺口 **20-30%** ✅（Lumentum CEO表示2028年产能已售罄）
- **合约情况**：云厂商通常Q4谈次年价格，涨价生效滞后 **1-2个季度** 🟡
- **竞争格局**：中际旭创/新易盛/光迅三家头部市占 **>60%**，具备一定协同定价能力 ✅
- **替代方案**：硅光方案占比从2024年35%提升至2026年 **60%**，缓解EML涨价压力 🟡

**综合评估**：当前转嫁率约 **60-80%** 🟡
- 即芯片涨价10% → 模块厂商内部消化20-40% → 终端售价实际上涨6-8%

### 3.4 时滞（Time Lag）

| 传导环节 | 时滞 | 原因 |
|---------|------|------|
| 原材料→芯片 | **1-2个季度** | 衬底采购周期 + 外延流片周期 + 库存缓冲 |
| 芯片→模块 | **1-2个季度** | 芯片交付周期 + 模块生产周期 + 库存缓冲 |
| 模块→终端售价 | **1-2个季度** | 合约谈判周期 + 招标流程 + 库存消化 |
| **全链条总时滞** | **3-6个季度** | 从原材料涨价到终端售价调整的完整周期 |

**特殊加速条件**：
- 若出现 **极端缺芯**（如2025Q4的200G EML），芯片厂商可要求 **即时调价** 或 **竞价分配**，时滞缩短至 **<1个季度** 🔴
- 若云厂商 **恐慌性囤货**，时滞进一步缩短 🔴

## 五、CCF 敏感性分析

### 5.1 上游衬底涨价10% 的全链条传导模拟

**假设条件**：
- InP衬底涨价 10%（如从$500→$550/片，4寸）
- 各环节按 **全额传导** 计算（理论最大值）
- 实际传导需乘以转嫁率

| 环节 | 成本变化 | 价格变化（理论） | 价格变化（实际，转嫁率60%） | 毛利率影响（若不自传导） |
|------|---------|----------------|------------------------|---------------------|
| InP衬底 | +10% | +10% | +10% | 维持 |
| 光芯片 | +0.15%（CCF_up≈0.015） | +0.15% | +0.09% | 挤压0.06pp |
| 光模块 | +2.2%（CCF_down≈14.9×0.15%） | +2.2% | +1.32% | 挤压0.88pp |
| 终端售价 | — | +1.32% | +1.32% | 云厂商成本↑ |

**关键洞察**：
- 衬底涨价10% → 芯片成本仅增0.15%（因芯片毛利率高、单模块衬底用量小）
- 但芯片若同步涨价（哪怕只涨1%）→ 模块成本增 **14.9%**（因芯片占模块BOM 35%+）
- **真正的传导放大器在中游→下游环节**，而非上游→中游

### 5.2 中游芯片涨价10% 的传导模拟

| 环节 | 成本变化 | 价格变化（理论） | 价格变化（实际，转嫁率60%） | 毛利率影响 |
|------|---------|----------------|------------------------|---------|
| 光芯片 | +10% | +10% | +10% | 维持 |
| 光模块 | +149%（CCF_down≈14.9） | +149% | +89.4% | 挤压59.6pp |

**现实修正**：
- 模块厂商不可能接受毛利率从40%→-20%
- 实际博弈结果：
  - 模块厂商向终端涨价 **30-50%**（部分传导）

## 六、CCF 动态修正因子

### 7.1 事件时间线

| 2026年Q2 | 源杰科技扩产公告（近20亿投资） | 国产替代加速 |

### 7.2 CCF复盘计算

**上游→中游（衬底→芯片）**：
- InP衬底涨幅：~200%（极端行情）
- 200G EML芯片涨幅：~30-50%（从$12→$18-22）🟡
- **实际CCF_up** = 50% / 200% = **0.25**（高于理论0.015，因缺货恐慌叠加）

**中游→下游（芯片→模块）**：
- 200G EML芯片涨幅：~50%
- 800G光模块涨幅：~20-30%（从$500→$600-650）🟡
- **实际CCF_down** = 30% / 50% = **0.6**（远低于理论14.9，因模块厂商压缩毛利+转向硅光）

**下游→终端（模块→云厂商）**：
- 800G模块涨幅：~30%
- 云厂商光互联成本涨幅：~10-15%（因模块占网络成本15-25%）🟡
- **转嫁率** = 15% / 30% = **0.5**（模块厂商未完全传导）

### 7.3 关键发现

1. **上游→中游传导被大幅稀释**：衬底涨200% → 芯片仅涨50%，芯片厂商高毛利提供了缓冲垫
2. **中游→下游传导被人为抑制**：理论CCF=14.9，实际仅0.6，模块厂商压缩自身毛利率+技术替代吸收了大部分冲击
3. **终端转嫁率约50%**：云厂商议价能力强，模块厂商无法全额传导
4. **时滞约2-3个季度**：从2025Q2原料涨价 → 2026Q1模块涨价生效
