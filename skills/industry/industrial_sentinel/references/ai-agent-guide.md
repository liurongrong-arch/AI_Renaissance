# Industrial Sentinel V4.5 — AI/Agent 自动执行指南

> **本文档写给使用本框架的 AI/Agent**
> 
> 如果你是被用户调用来执行产业链分析的 AI，请按以下步骤自动完成从数据搜索到报告生成的全流程。

---

## 你的任务

用户输入一个股票代码（如 `688XXX.SH`），你的任务是：
1. **自动搜索数据**（按本清单）
2. **填入JSON模板**
3. **调用框架推理**
4. **返回HTML报告**

用户不需要手动做任何事。你全程自动执行。

## Step 2: 搜索数据（你必须执行）

根据检测到的产业链，按以下清单搜索数据。每个数据点必须标注**来源**和**时间戳**。

### 2.1 财报数据（最高优先级）

搜索查询模板（用你最擅长的搜索工具执行）：

```
搜索1: "{stock_code} Q1 2026 财报 营收 同比"
搜索2: "{stock_code} 2026年一季度 毛利率 净利润"
搜索3: "{stock_name} earnings Q1 2026 revenue gross margin"（美股）
搜索4: "{stock_code} 订单 backlog pipeline 业绩会"
搜索5: "{stock_code} 产能利用率 扩产 资本开支"
搜索6: "{stock_code} 库存 库存天数 周转"
```

**必填字段**（至少搜到3项）：

| 字段 | 搜索关键词 | 填入格式 |
|------|-----------|---------|
| 营收增速 | "Q1 2026 营收 同比" | `45.2` |
| 毛利率 | "毛利率 最新季度" | `25.3` |
| 订单backlog | "订单 backlog 业绩会" | `"$100M+"` 或 `"饱满（定性）"` |
| 产能利用率 | "产能利用率 扩产" | `85` 或 `"扩产中"` |
| 产品价格 | "产品价格 涨价" | `"$2500/片"` |
| 库存天数 | "库存天数 周转" | `45` |
| 扩产计划 | "资本开支 capex" | `"扩产50%，2026Q3投产"` |
| 政策数量 | "政策 补贴 准入" | `2` |

### 2.2 行业数据（至少搜5项）

## 常见问题处理

### Q1: 搜不到某条数据怎么办？
**处理**：标注"数据缺失"，继续执行。框架会降级输出，不会报错。

### Q2: 不同来源数据矛盾怎么办？
**处理**：优先采信财报（一手来源），标注矛盾并说明采信理由。

### Q3: 定性描述无法量化怎么办？
**处理**：保留定性描述，但标注"（定性）"，框架会按定性规则处理。

### Q4: 搜索超时或API失败怎么办？
**处理**：重试1次，仍失败则标注"搜索失败"，用已有数据继续分析。
